# 终末地副本推荐

一个基于C#的命令行工具，用于为《明日方舟·终末地》中的武器推荐最优的副本和词条指定方式。

如果掉落的词条有3、2、1个词条和武器词条一致，则输出为完美、良好、一般掉落。输出结果必然包含用户指定的武器的完美掉落概率，并最大化其他武器的完美掉落期望（次要优化目标为良好掉落和一般掉落期望）。如果已经有一个武器的完美基质胚子，可以将它从`武器.txt`里增加前缀`#`，这样就不会再纳入计算了。

## 前置要求

- .NET 8.0 SDK 或更高版本

## 运行步骤

1. 确保项目目录下存在以下数据文件：
   - `武器.txt` - 武器数据文件
   - `副本.txt` - 副本数据文件

2. 编译并运行程序：
   ```bash
   dotnet build
   dotnet run
   ```

3. 根据提示输入武器名称，程序将输出最优推荐方案。

## 示例

```
请输入武器名：
沧溟星梦

推荐方案：
================================================================================
副本：供能高地
词条指定：主词条 [主、敏、智]，技能词条 [附]
完美掉落：【沧溟星梦（4.2%）】、【宏愿（4.2%）】、【黯色火炬（4.2%）】...
```
